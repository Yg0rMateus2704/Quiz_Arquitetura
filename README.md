# 🎮 Jogo Educativo de Arquitetura com Labirinto (p5.js)

Este projeto é um **jogo interativo desenvolvido em JavaScript utilizando a biblioteca p5.js**, que combina **educação e entretenimento**.

O jogador avança por diferentes fases relacionadas a estilos arquitetônicos, respondendo perguntas e enfrentando desafios em um **labirinto controlado pelo mouse**.

---

## 🧠 Objetivo

O objetivo do jogo é:

- Ensinar sobre diferentes **estilos arquitetônicos**
- Testar o conhecimento do jogador com **perguntas de múltipla escolha**
- Oferecer **dicas desbloqueáveis** através de desafios
- Proporcionar uma experiência divertida com mecânicas de jogo simples

---

## 🎮 Como Jogar

1. Inicie o jogo (tela inicial)
2. Escolha entre:
   - ▶️ Jogar
   - 📖 Ver instruções
3. Responda corretamente às perguntas
4. Caso erre:
   - ❌ Game Over
5. Para obter dicas:
   - Complete o desafio do labirinto
6. Avance por várias fases até chegar à vitória 🎉
